This program is a Distributed Social (DS) profile manager that allows users to create, edit, and manage DSU profiles and journal posts. Users can create profiles, add or delete posts, view profile information, and publish posts to the DS server. The program provides both a friendly user interface and an admin mode for interacting with profiles and files.

Assignment 4 extends the DS platform by integrating two web APIs: OpenWeather and LastFM. When users write posts, they can include special keywords such as @weather and @lastfm. The program retrieves data from the respective APIs and replaces these keywords with real-time information, such as the current weather description or the top music artist. The API functionality is implemented using reusable classes that inherit from a base WebAPI class, which handles shared behavior for connecting to web APIs.
